NgIf

Uma diretiva estrutural que inclui condicionalmente um modelo baseado no valor de uma expressão coagida para Boolean. Quando a expressão é avaliada como verdadeira, o Angular renderiza o modelo fornecido em uma thencláusula e, quando falso ou nulo, o Angular renderiza o modelo fornecido em uma elsecláusula opcional. O modelo padrão para a elsecláusula está em branco.

Uma forma abreviada da diretiva, , é geralmente usada, fornecida como um atributo do elemento âncora para o modelo inserido. Angular expande isso para uma versão mais explícita, na qual o elemento âncora está contido em um elemento.*ngIf="condition"
